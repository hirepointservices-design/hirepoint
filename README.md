## What's included
- React + Vite app with Tailwind
- Navbar, Hero slider (framer-motion), Services, About, Contact, Footer
- Placeholder slide images in public/assets/
- Express backend (server/index.js) to capture leads to server/leads.json

## Run locally
1. Install Node.js (v16+ recommended)
2. npm install
3. npm run dev   (starts Vite at http://localhost:5173)
4. In another terminal: npm run server  (starts backend at http://localhost:5000)

## Replace content

- Replace images in public/assets/ with your real images (keep same filenames or update references).
